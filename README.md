# Recipe Explorer


A fullstack web app to explore meals and recipes from around the world using TheMealDB API.


## Tech stack

### Backend

* Nest.js
* Typescript
* Axios
* TheMealDB API

### Frontend

* Next.js
* React
* TypeScript

## Setup

First of all, clone or download this repository

### Backend setup

1. Navigate to the backend folder:

`cd backend`

2. Install dependencies

`npm install`

3. Start the server

`npm run start:dev`

4. App will run on http://localhost:3001/

### Frontend setup 

1. Navigate to the backend folder:

`cd front_end`

2. Install dependencies

`npm install`

3. Start the app

`npm run dev`

4. App runs on http://localhost:3000/
